# Configuration Guide

The config file supports three formats: YAML, JSON, and TOML.

Pick whichever you prefer. I use YAML because it's easier to read,
but JSON works fine if you're generating config programmatically.

Here's the tricky part -- the env var overrides only work for
top-level keys. Nested config requires a file.
